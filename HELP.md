# Getting Started

JAVA筆試題目：
1. 設計一個計算機（Calculator），可以執行兩個數的加、减、乘、除計算，可
   以進行undo和redo操作，在實現功能的基礎上進行最佳化設計，備註：undo
   和redo 就是 復原和取消(上次)復原的操作
2. 設計一個電商轉盤抽獎功能，3種獎品各有N種數量且每個獎品的中獎機率
   不同，與銘謝惠顧合起來機率為100%，可有同時多次抽獎的機會並包含防
   止重複抽獎與獎品超抽的情況。
